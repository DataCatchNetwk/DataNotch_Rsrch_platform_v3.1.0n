# Final Dataset Page Pack

Production-style starter pack for a research-platform **Datasets** page using:
- Next.js App Router
- shadcn/ui patterns
- TypeScript
- Tailwind
- domain filters, search, favorites, preview modal, pull modal
- grid/table toggle
- workspace pull wiring

## Included
- `apps/web/src/app/(dashboard)/datasets/page.tsx`
- dataset UI components
- typed API client
- shared dataset types

## Notes
- Update import aliases if your project uses a different `@/` setup.
- Replace placeholder shadcn imports with your existing project components.
- Wire the API base URL to your environment.
