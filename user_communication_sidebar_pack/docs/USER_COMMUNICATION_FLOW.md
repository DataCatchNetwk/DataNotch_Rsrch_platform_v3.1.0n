# User Communication Flow

Inbox = human communication.
Notifications = system events.
Messages = real-time/user-created conversations.
Meetings = R-MEET and R-ZOOMA sessions.
Tasks = meeting-generated or research-workflow assignments.

Research Asset Messaging:
Project -> Messages
Study -> Messages
Dataset -> Messages
Analysis -> Messages
Publication -> Messages

Example:
Dataset: Clinical_SDOH_v5
- Admin asks for missingness review
- Researcher replies with validation notes
- Data Steward approves after harmonization
