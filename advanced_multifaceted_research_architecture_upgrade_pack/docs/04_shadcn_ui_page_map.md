## Recommended page modules
### Data Deposit
- table + card toggle
- metadata drawer
- ontology tags
- favorites and saved filters
- pull to workspace modal

### Cohort Builder
- criteria builder
- domain/time window selectors
- reusable phenotype blocks
- cohort preview counts
- save as template

### Feature Store
- feature recipes
- derivation graph
- validation status
- reuse in analysis

### Analysis Studio
- pipeline launcher
- parameter panels
- model selection
- metrics tabs
- artifact viewer

### Experiments
- run comparison table
- champion/challenger view
- metric trend chart
- dataset/feature version binding

### Genomics
- omics dataset registration
- sample/variant summary
- pathway enrichment result viewer
