# Capability Matrix

| Area | MVP | Growth | Advanced |
|---|---|---|---|
| Discovery | Domain filters, search | Semantic search, ontology tags | Relevance ranking, concept expansion |
| Cohorts | Basic filters | Temporal criteria, saved cohorts | Computable phenotypes, reusable cohort templates |
| Features | Manual variables | Derived feature recipes | Versioned feature store with validation |
| Statistics | Descriptive stats, linear/logistic regression | Mixed models, ANOVA, survival summaries | Advanced causal and longitudinal analysis |
| Machine Learning | Classification/regression | CV, tuning, feature importance | AutoML, drift checks, fairness diagnostics |
| Survival | KM curves | Cox PH, concordance | Competing risks, time-varying covariates |
| Genomics | Metadata registration | Expression/VCF ingestion | Variant pipelines, enrichment, GWAS-ready flows |
| Evaluation | Accuracy, precision, recall, F1 | ROC/PR AUC, calibration | Fairness and robust uncertainty analysis |
| Workspaces | Project datasets | Frozen snapshots | Reproducible environments and publication bundles |
| Collaboration | Workspace members | Review, approvals | Shared templates, peer-review exports |
