# Analysis Engine Design

## Statistics Engine
Supported jobs:
- descriptive statistics
- linear regression
- logistic regression
- multinomial regression
- ANOVA
- chi-square
- mixed-effects models
- imputation routines

## Machine Learning Engine
Supported jobs:
- classification
- regression
- clustering
- dimensionality reduction
- feature selection
- cross-validation
- tuning
- explainability reports

## Survival Engine
Supported jobs:
- Kaplan-Meier curves
- log-rank tests
- Cox proportional hazards
- hazard summaries

## Genomics Engine
Supported jobs:
- variant metadata indexing
- expression matrix normalization
- pathway enrichment
- differential analysis scaffolds

## Evaluation Engine
Metrics supported:
- accuracy
- precision
- recall
- F1-score
- ROC-AUC
- PR-AUC
- RMSE
- MAE
- R²
- concordance index
