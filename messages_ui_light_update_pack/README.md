# Research Platform V3 Messages UI Light Update

This pack replaces the previous dark prototype with a light, purple-accented messaging page matching the uploaded reference image.

## File
- `app/messages/page.tsx`

## Features
- Full Research Platform V3 shell
- Left icon rail
- Labeled communication sidebar
- Top search/user header
- Inbox, All, Unread, Starred tabs
- Searchable thread list
- Active thread message view
- Reply composer with actions
- Thread details panel
- Participants
- Attachments
- Associated research asset
- Actions: mark unread, archive, mute

## Install
Copy `app/messages/page.tsx` into your Next.js App Router project.

Required packages:
```bash
npm install lucide-react
```

Tailwind CSS must already be configured.
