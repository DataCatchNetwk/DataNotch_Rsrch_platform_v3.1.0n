# Algorithmic Logic Included

This pack includes deterministic, dependency-light engines suitable for integration and demo:

- Mean, median, mode, variance, standard deviation, quantiles
- Group comparisons and disparity-rate inference
- Correlation-based ML feature importance
- ROC-ready classification output
- SHAP/LIME-style explainability fallback
- Causal treatment-control ATE simulation
- Kaplan-Meier style survival curve data
- Time-series forecast/trend data
- Network centrality and graph output
- Choropleth-ready geographic summaries
- Knowledge graph path reasoning
- Digital twin intervention simulation
- Counterfactual scenario ranking

Production upgrades can replace these engines with:
- Python/R workers
- scikit-learn
- xgboost/lightgbm
- lifelines
- statsmodels
- DoWhy/EconML
- PyTorch
- Neo4j GDS
- GeoPandas/PostGIS
