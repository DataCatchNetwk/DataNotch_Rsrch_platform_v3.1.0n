# User Communication Sidebar Pack

Research Platform V3 user-facing communication module.

Includes:
- User Communication Sidebar
- Inbox, Notifications, Meetings, Messages, Tasks, Invitations, Announcements, Support Center, History
- Research asset discussions for Project, Study, Dataset, Analysis, Publication
- API routes and service logic
- Prisma schema models

## Frontend
Copy `frontend/components/UserCommunicationSidebar.tsx` and `frontend/pages/UserCommunicationHub.tsx` into your Next.js app.

## Backend
Copy `backend/src/userCommunication.routes.ts` and `backend/src/userCommunication.service.ts` into your API server.

## Prisma
Merge `backend/prisma/user_communication.prisma` into your existing Prisma schema.
