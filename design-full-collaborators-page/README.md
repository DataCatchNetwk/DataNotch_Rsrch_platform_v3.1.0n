# Design Full Collaborators Page

This downloadable pack contains a polished shadcn-based Global Collaborators page for a research platform.

## Included
- Summary stats cards
- Invite collaborator modal
- Advanced filters
- Rich collaborators table
- Pending invites side panel
- Collaborator detail drawer
- Mock data + types

## Expected shadcn components already in your project
- avatar
- badge
- button
- card
- dialog
- dropdown-menu
- form
- input
- select
- separator
- sheet
- table
- textarea

## Suggested next step
Wire the mock data to your real NestJS endpoints for:
- GET /collaborators
- GET /collaborators/stats
- GET /collaborators/invites
- POST /collaborators/invite
- PATCH /collaborators/:id/role
- POST /collaborators/:id/revoke
